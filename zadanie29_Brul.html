Koniec pytań teoretycznych

1. Określanie właściwości danego obiektu

Sposób 1 – zapis z kropką


nazwa_obiektu.nazwa_właściwości = "tekst";
nazwa_obiektu.nazwa_właściwości = liczba;

Przykład:


auto.marka = "Opel Omega";
auto.rok = 1996;
auto.cena = 25000;

Sposób 2 – zapis tablicowy


nazwa_obiektu["nazwa_właściwości"] = wartość;

Przykład:


auto["cena"] = 25000;

2. Odwoływanie się do pól (właściwości) i metod

Do pól i metod obiektu odwołujemy się poprzez nazwę obiektu oraz kropkę. 


console.log(samochod1.kolor);
samochod1.jedz();

3. Cechy charakterystyczne konstruktora

  • Jest wywoływany automatycznie podczas tworzenia obiektu.
  • Nazywa się tak samo jak klasa (funkcja konstruktora).
  • Nie zwraca żadnej wartości.
  • Może przyjmować argumenty.
  • Do jego użycia stosujemy słowo kluczowe new.

4. Konstruowanie obiektu – trzy pola i jedna metoda


var osoba_1 = {
  nazwisko: "Kowalski",
  imie: "Jan",
  zawod: "piekarz",
  wyswietl: function() {
    document.write(this.nazwisko + " " + this.imie);
  }
}

5. Znaczenie słowa this

Słowo kluczowe this pozwala odwołać się do aktualnego obiektu z wnętrza tego obiektu.

Pola i metody w obiekcie oddzielamy przecinkami.

6. Konstruktor w JavaScript – przykład i opis


function klient(nazwisko_k, imie_k, zawod_k) {
  this.nazwisko = nazwisko_k;
  this.imie = imie_k;
  this.zawod = zawod_k;
  this.wypisz = function() {
    alert(this.nazwisko + " " + this.imie);
  }
}

Funkcja pełni rolę konstruktora. Słowo this odnosi się do aktualnie tworzonego obiektu.

7. Słowo kluczowe new

Słowo kluczowe new służy do tworzenia nowego obiektu na podstawie konstruktora. 


var osoba1 = new klient("Kowalski", "Jan", "kierowca");
var osoba2 = new klient("Nowak", "Anna", "sekretarka");

Powstają dwa niezależne obiekty należące do konstruktora klient.

8. Właściwość prototype

Prototype umożliwia dodawanie metod i właściwości wspólnych dla wszystkich obiektów utworzonych na podstawie danego konstruktora. 


function klient() {
  this.nazwisko = "Bielski";
  this.imie = "Paweł";
}

klient.prototype.pisz_dane = function() {
  document.write(this.nazwisko + " " + this.imie);
}

klient.prototype.zawod = "kierowca";

var osoba1 = new klient();
osoba1.pisz_dane();

Po użyciu prototype każdy nowy obiekt konstruktora klient posiada dodane metody i właściwości.
