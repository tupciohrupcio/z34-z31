<p class="pytanie">1. Cztery sposoby wyprowadzania danych w JS</p>
- document.write()
- alert()
- wpisywanie do pól formularza
- wpisywanie danych do tabeli HTML

<p class="pytanie">2. Funkcja w JS</p>
- Zbiór instrukcji wykonywanych po wywołaniu
- Realizuje kod wewnętrzny
- Może mieć argumenty np. suma(a,b)
- Może zwrócić wynik przez return

<p class="pytanie">3. Metody JS</p>
getElementById() – dostęp po id
getElementsByTagName() – dostęp po znaczniku
getElementsByName() – dostęp po name

<p class="pytanie">4. Właściwości</p>
innerHTML  – zarządza kodem HTML w elemencie
innerText   – zarządza tekstem bez HTML
className   – ustawia klasę CSS
